Form Validation

To help future interested employers, I have added a contact form to my profile page. The purpose is to help them have an easy way to leave their contact details and any information on why they want to get in touch.

I used HTML and CSS to create and style the page respectively.

I used JavaScript to help me record user events and change elements. This included ensuring there were no blank fields and customized error messages popped up when a user left a field blank. It also prevents the user from submitting an uncompleted form. HTML does have error messages, but I wanted something that would explain why the user couldn't submit the form.

The code below show a function I created to check the validity of an email address.

// Function to check validation of email
function isValidEmail(email) {
return /^[A-Za-z][\._\-][0-9]*[@][A-Za-z]*[\.][a-z]{2,4}$/.test(String(email).toLowerCase()); }
